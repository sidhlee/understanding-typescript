### Type Assignment and Type Inference

You can manually assign types with `let` (not `const`)

```ts
let number1: number;
number1 = 5;
```

Or you can let TS to inference type automatically

```ts
let resultPhrase = 'Result is: ';
// resultPhrase = 0; // ts yells error
```
